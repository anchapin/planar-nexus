Create a hook that calculates deck statistics from a deck of cards:

1. Define CardAnalysisResult interface:
```typescript
export interface CardAnalysisResult {
  totalCards: number;        // Total card count (sum of all counts)
  uniqueCards: number;       // Number of unique cards
  colorDistribution: Record<string, number>;  // W, U, B, R, G, colorless
  manaCurve: Record<number, number>;  // 0-7+ CMC distribution
  typeDistribution: Record<string, number>;  // creature, instant, sorcery, etc.
  averageManaValue: number;
}
```

2. Implement analyzeDeck function:
- Input: DeckCard[] (with count property)
- Output: CardAnalysisResult

3. Calculate:
- **Mana curve**: For each card, get cmc (converted mana cost), add card count to that CMC bucket. Handle lands (CMC 0) and cards with CMC >= 7 in overflow bucket.
- **Color distribution**: For each card, determine its colors from the colors array. Count cards by color (W, U, B, R, G, Colorless). Account for multi-color cards.
- **Type distribution**: Parse type_line to extract types (creature, instant, sorcery, enchantment, artifact, planeswalker, land). Count by type.
- **Average CMC**: Sum of (card.cmc * card.count) / totalCards

4. Export useMemo-wrapped version for React use
