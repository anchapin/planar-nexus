Extend filter-cards.ts with color filtering functions:

1. `filterByColor(cards: MinimalCard[], filter: ColorFilter): MinimalCard[]`
   - exact mode: card.colors must exactly match filter.colors (including colorless)
   - include mode: card must have ALL filter.colors (can have others)
   - exclude mode: card must NOT have ANY of filter.colors

   - Same logic as filterByColor but uses color_identity field
   - This respects cards that produce colors through abilities

3. `filterByCommanderRules(cards: MinimalCard[], allowedColors: string[]): MinimalCard[]`
   - Apply Commander color identity rules:
   - A card is legal if its color_identity is a SUBSET of allowedColors
   - For 5-color commander (WUBRG), allow all cards
   - For 2-color (e.g., UW), allow cards that only have U and/or W
   - Use the COMMANDER_COLOR_IDENTITY_RULES from game-rules.ts

**Helper:** Create `isColorSubset(cardColors: string[], allowedColors: string[]): boolean`
   - Returns true if every color in cardColors is in allowedColors
   - Empty cardColors (colorless) returns true

Import ColorFilter from filter-types.ts.
