<task id="07-04-02" title="Implement Background Indexing Manager">
<files>
  <file>src/lib/search/background-indexing.ts</file>
</files>
<action>
Create `src/lib/search/background-indexing.ts`:
- Create a `BackgroundIndexingManager` class to orchestrate the process:
  1. Retrieve cards from the source database.
  2. Check `embeddings` table for existing vectors.
  3. Send missing cards to the `EmbeddingClient` in batches.
  4. Store generated embeddings back into Dexie.js.
  5. Upsert cards into the `OramaManager`.
  6. Trigger `saveIndex()` once indexing is complete or periodically.
- Implement delta indexing: only process cards added/updated since the last run.
</action>
<verify>
Check the orchestration flow for correctness and handling of large card datasets without crashing.
</verify>
<done>
Background indexing orchestration is implemented.
</done>
</task>
