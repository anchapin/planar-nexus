    1. Accepts `deck: DeckCard[]` as a prop.
    2. Manages a `Worker` instance for `src/lib/ai/embedding-worker.ts`.
    3. Implements a debounced effect (500ms) that sends the current deck list to the worker to generate a composite "deck vector".

    - Handle worker messages (`EMBEDDINGS_GENERATED`, `ERROR`).
    - The "deck vector" should be the mean of all card embeddings in the deck (or use the worker to compute it if it supports batching).
  </action>
