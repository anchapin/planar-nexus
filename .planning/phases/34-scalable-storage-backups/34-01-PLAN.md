Implement incremental backup and restore logic in IndexedDBStorage.

Purpose: The current exportBackup() dumps all data as a single JSON blob. For 50MB+ collections this is slow and wasteful. Incremental backup exports only records modified after a given timestamp, and incremental restore merges (upserts) rather than replacing all data.
Output: New types (IncrementalBackupData, BackupManifest) and two new methods on IndexedDBStorage.
</objective>

      since: string;

      savedGames: StoredGame[];
      deletedRecords: Array<{ store: string; id: string }>;
      checksum: string;

    export interface BackupManifest {
      id: 'backup-manifest';
      lastFullBackupAt: string | null;
      lastIncrementalBackupAt: string | null;

    }

    2. Add `exportIncrementalBackup(since: Date): Promise<IncrementalBackupData>` to IndexedDBStorage:

    - For each savedGame in data.savedGames: call `this.set('saved-games', savedGame)`
    - For each entry in data.deletedRecords: call `this.delete(entry.store, entry.id)`

    4. Add `getBackupManifest(): Promise<BackupManifest | null>`:
