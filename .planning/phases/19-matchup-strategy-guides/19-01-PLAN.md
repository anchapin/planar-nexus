# Phase 19 Plan 1: Anti-Meta Counter Recommendations

**Phase**: 19 - Matchup & Strategy Guides  
**Subsystem**: Meta Strategy  
**Priority**: High  
**Dependencies**: Phase 18 (completed)

---

## Context

Users need to counter popular meta decks. This plan implements anti-meta recommendations including counter decks, sideboard suggestions, and mana base recommendations.

## Requirements Covered

| ID | Requirement | Description |
|----|-------------|-------------|
| AMETA-01 | Anti-deck recommendations | User can get counter recommendations for specific deck archetypes |
| AMETA-02 | Sideboard recommendations | User can see sideboard recommendations vs. popular matchups |
| AMETA-03 | Mana base recommendations | User can view mana base recommendations for anti-meta decks |

## Implementation Plan

### Task 1: Create Anti-Meta Data Types & Mock Data

**File**: `src/lib/anti-meta.ts`

- Create `CounterRecommendation` interface:
  - `archetypeId: string` - target archetype being countered
  - `counterArchetypeId: string` - recommended counter deck
  - `counterArchetypeName: string` - display name
  - `matchupWinRate: number` - expected win rate
  - `keyCards: string[]` - essential cards for the counter
  - `sideboardNotes: string` - sideboard strategy

- Create `AntiMetaService` with mock data:
  - 3-5 counter recommendations per format (Standard, Modern, Commander)
  - Link to existing archetypes from meta.ts

- Helper functions:
  - `getCounterRecommendations(archetypeId, format)`
  - `getSideboardRecommendations(archetypeId, format)`
  - `getManaBaseRecommendations(archetypeId, format)`

### Task 2: Create CounterDeckCard Component

**File**: `src/components/meta/CounterDeckCard.tsx`

- Props: `CounterRecommendation`
- Display:
  - Counter deck name with archetype badge
  - Matchup win rate (color-coded)
  - Key cards preview (top 3)
  - "View Details" button
- Styling: Use existing Card, Badge components from shadcn

### Task 3: Create AntiMetaRecommendations Component

**File**: `src/components/meta/AntiMetaRecommendations.tsx`

- Input: Selected archetype from meta page
- Sections:
  1. "Counters to [Archetype Name]" - list of CounterDeckCard
  2. Sideboard guide summary
  3. Mana base tips
- Integration: Connects to existing MetaOverview

### Task 4: Integrate into Meta Page

**File**: `src/app/(app)/meta/page.tsx`

- Add "Get Counter Advice" button on each DeckArchetypeCard
- On click, show AntiMetaRecommendations in a Dialog/Sheet
- Update sidebar to show Meta page is complete
