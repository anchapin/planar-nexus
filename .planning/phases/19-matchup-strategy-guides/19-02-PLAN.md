# Phase 19 Plan 2: Matchup Strategy Guides

**Phase**: 19 - Matchup & Strategy Guides  
**Subsystem**: AI Strategy  
**Priority**: High  
**Dependencies**: Phase 18 (completed), Plan 19-01

---

## Context

Users need pre-game strategic advice specific to their deck vs. opponent archetype matchups. This plan implements matchup guides with mulligan recommendations and game plan tips.

## Requirements Covered

| ID | Requirement | Description |
|----|-------------|-------------|
| MATCH-01 | Pre-game strategy | User can get pre-game strategic advice vs. specific archetypes |
| MATCH-02 | Mulligan recommendations | User can see mulligan recommendations per matchup |
| MATCH-03 | Game plan tips | User can view game plan tips (aggro vs. control vs. midrange) |

## Implementation Plan

### Task 1: Create Matchup Data Types & Service

**File**: `src/lib/matchup-guides.ts`

- Create `MatchupGuide` interface:
  - `playerArchetype: string` - user's deck type
  - `opponentArchetype: string` - AI/opponent deck type
  - `winRate: number` - expected win rate
  - `gamePlan: GamePlanTips` - strategic advice
  - `mulliganGuide: MulliganTips` - mulligan recommendations
  - `keyCards: string[]` - important cards for the matchup

- Create `GamePlanTips`:
  - `opening: string[]` - opening phase priorities
  - `midGame: string[]` - mid-game priorities
  - `lateGame: string[]` - late-game priorities
  - `generalStrategy: string` - overall approach

- Create `MulliganTips`:
  - `keep: string[]` - cards worth keeping
  - `mulligan: string[]` - cards to ship
  - `consider: string[]` - situational keeps
  - `notes: string` - general mulligan strategy

- Create mock data:
  - Guides for all archetype pairings (aggro, control, midrange, combo, tempo vs. each other)
  - 15+ matchup combinations per format

- Helper functions:
  - `getMatchupGuide(playerArchetype, opponentArchetype)`
  - `getMulliganGuide(playerArchetype, opponentArchetype)`
  - `getGeneralStrategy(playerArchetype)`

### Task 2: Create MatchupGuideCard Component

**File**: `src/components/meta/MatchupGuideCard.tsx`

- Props: `MatchupGuide`
- Display:
  - Matchup header: "Your [Archetype] vs [Opponent Archetype]"
  - Win rate badge
  - Collapsible sections for each game phase
  - Mulligan tips in a special callout box
- Uses Accordion component from shadcn

### Task 3: Create MulliganTips Component

**File**: `src/components/meta/MulliganTips.tsx`

- Props: `MulliganTips`
- Display:
  - Keep (green), Mulligan (red), Consider (yellow) sections
  - Visual card icons for each category
  - Notes section at bottom
- Uses Card, Badge components

### Task 4: Create MatchupSelector Component

**File**: `src/components/meta/MatchupSelector.tsx`

- Two dropdowns: Player Archetype, Opponent Archetype
- Format selector (Standard/Modern/Commander)
- "Get Guide" button
- Shows recent matchups (from localStorage)

### Task 5: Create MatchupGuidePage

**File**: `src/app/(app)/matchup/page.tsx`

- Full page for matchup guides
- MatchupSelector at top
- MatchupGuideCard below
- "Save to Deck" CTA
- Add to sidebar navigation
