### Task 2.2.2: Define 15+ Archetype Signatures
**Type**: auto
**Duration**: ~60 min

**Actions**:
1. Create archetype signature definitions:

**Aggro (3)**:
- Burn: Direct damage ≥40%, low curve (avg CMC ≤2.5)
- Zoo: Creatures ≥60%, pump spells ≥15%, low curve
- Sligh: Creatures ≥70%, very low curve (avg CMC ≤2.0)

**Control (3)**:
- Draw-Go: Counterspells ≥20%, card draw ≥15%, few creatures
- Stax: Artifacts ≥30%, land destruction ≥10%, high curve
- Prison: Lock pieces ≥15%, removal ≥20%, high curve

**Midrange (3)**:
- Good Stuff: Balanced curve, threats ≥40%, answers ≥30%
- Rock: Creatures ≥45%, removal ≥20%, ramp ≥15%
- Value: Card advantage ≥20%, efficient creatures ≥35%

**Combo (3)**:
- Storm: Cantrips ≥30%, ritual effects ≥10%, combo pieces
- Reanimator: Reanimation ≥15%, graveyard setup ≥20%
- Infinite: Combos ≥20%, tutors ≥15%, protection ≥15%

**Tribal (4)**:
- Elves: Elf creatures ≥60%, lords ≥15%, ramp ≥15%
- Goblins: Goblin creatures ≥60%, lords ≥15%, sacrifice ≥10%
- Zombies: Zombie creatures ≥60%, graveyard ≥20%
- Dragons: Dragon creatures ≥40%, ramp ≥20%, high curve

**Special (2)**:
- Lands: Lands ≥50%, land recursion ≥20%
- Superfriends: Planeswalkers ≥25%, protection ≥25%

2. Define detection thresholds for each
3. Document in `src/ai/archetype-signatures.ts`

**Verification**:
- Each archetype has clear signature
- Signatures are mutually exclusive where possible
- Edge cases documented

### Task 2.2.3: Implement Archetype Detection Algorithm
**Type**: auto
**Duration**: ~90 min

**Actions**:
1. Create/update `src/ai/archetype-detector.ts`:
```typescript
interface ArchetypeSignature {
  name: string;
  minCreatureRatio?: number;
  maxCreatureRatio?: number;
  minSpellRatio?: number;
  landRatio?: [number, number]; // [min, max]
  avgCmcRange?: [number, number];
  keywordRequirements?: Map<string, number>;
  cardTypeRequirements?: Map<string, number>;
  scoreFunction: (deck: DeckCard[]) => number;
}

export function detectArchetype(deck: DeckCard[]): ArchetypeResult;
```

2. Implement scoring algorithm:
   - Calculate deck ratios (creatures, spells, lands)
   - Calculate mana curve statistics
   - Count keywords (flying, trample, etc.)
   - Count card types (planeswalkers, artifacts, etc.)
   - Score against each archetype signature
   - Return best match with confidence score

3. Handle multi-archetype decks (return top 2 with scores)

**Verification**:
```typescript
// Test with known decks
const burnDeck = [...]; // 40% burn spells
const result = detectArchetype(burnDeck);
expect(result.primary).toBe('Burn');
expect(result.confidence).toBeGreaterThan(0.7);
```
