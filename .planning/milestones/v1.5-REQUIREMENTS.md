# v1.5 Requirements: Meta & Strategy AI

## Requirements Overview
This document defines the requirements for v1.5 Meta & Strategy AI milestone.

## Meta Analysis Features

### META-01: Meta Deck Analysis
**Priority:** HIGH

**Description:** Analyze top decks in selected format, identify dominant strategies.

**Acceptance Criteria:**
- [ ] User can select format (Standard, Modern, Commander, etc.)
- [ ] Display top 10 deck archetypes with win rates
- [ ] Show deck archetype card composition
- [ ] Identify key cards and synergies in each archetype
- [ ] Real-time meta data refresh capability

### META-02: Anti-Meta Recommendations
**Priority:** HIGH

**Description:** Suggest counters to popular deck types based on meta share.

**Acceptance Criteria:**
- [ ] For each popular deck type, show 2-3 counter strategies
- [ ] Provide win rate projections vs. popular matchups
- [ ] Suggest sideboard cards for anti-meta strategies
- [ ] Rank counters by effectiveness

### META-03: Format Health Score
**Priority:** MEDIUM

**Description:** Calculate and display format diversity metrics.

**Acceptance Criteria:**
- [ ] Calculate diversity index (0-100) based on deck distribution
- [ ] Show top deck percentage share
- [ ] Display number of viable archetypes
- [ ] Historical comparison to previous weeks
- [ ] Visual indicators (healthy/warning/unhealthy)

### META-04: Meta Trend Tracking
**Priority:** MEDIUM

**Description:** Show rising/falling deck archetypes over time.

**Acceptance Criteria:**
- [ ] Track deck archetype win rates over 7/30/90 days
- [ ] Highlight trending up/down archetypes
- [ ] Show percentage change in play rate
- [ ] Visual charts for trend visualization

## Strategic AI Features

### STRAT-01: Matchup Guides
**Priority:** HIGH

**Description:** Pre-game strategic advice vs. specific opponent archetypes.

**Acceptance Criteria:**
- [ ] User selects opponent deck archetype
- [ ] Display general strategy overview
- [ ] Show key cards to counter
- [ ] Mulligan recommendations
- [ ] Game plan for each phase

### STRAT-02: Sideboard Plans
**Priority:** HIGH

**Description:** Detailed sideboard recommendations per matchup.

**Acceptance Criteria:**
- [ ] Input: user deck + opponent deck archetype
- [ ] Generate sideboard plan (cards in/out)
- [ ] Explain reasoning for each change
- [ ] Show expected win rate improvement
- [ ] Support multiple matchup scenarios

### STRAT-03: Mana Curve Optimization
**Priority:** MEDIUM

**Description:** Suggest optimal land counts based on deck strategy.

**Acceptance Criteria:**
- [ ] Analyze deck's average CMC
- [ ] Calculate recommended land count (17-27 range)
- [ ] Suggest mana rock / artifact counts
- [ ] Factor in deck speed (aggro/control/midrange)
- [ ] Provide reasoning based on probability

### STRAT-04: Game Phase Strategy
**Priority:** MEDIUM

**Description:** Strategic advice for each game phase (opening, mid, late).

**Acceptance Criteria:**
- [ ] Opening: hand evaluation, keep/snap decisions
- [ ] Mid-game: board control, card advantage
- [ ] Late-game: topdeck wars, closing out
- [ ] Context-aware based on current game state
- [ ] Probability-based decision suggestions

## Integration Features

### INT-01: Meta Dashboard
**Priority:** HIGH

**Description:** New dashboard page showing meta overview.

**Acceptance Criteria:**
- [ ] Dedicated page at /meta or /dashboard/meta
- [ ] Format selector
- [ ] Quick stats overview
- [ ] Link to detailed analysis
- [ ] Refresh capability

### INT-02: Deck Comparison
**Priority:** MEDIUM

**Description:** Compare your deck to meta decks.

**Acceptance Criteria:**
- [ ] Load user's saved deck
- [ ] Compare to selected meta deck
- [ ] Highlight similarities and differences
- [ ] Suggest improvements
- [ ] Show matchup probabilities

### INT-03: Meta Export
**Priority:** LOW

**Description:** Export meta reports for sharing.

**Acceptance Criteria:**
- [ ] Export as JSON/CSV
- [ ] Include deck lists
- [ ] Timestamp for historical reference
- [ ] Shareable link generation
