## Tasks

### Task 1: Create Mana Curve Data Types
**Action**: Create `src/lib/mana-curve.ts`
- TypeScript interfaces:
  - `ManaCurvePoint` - CMC to count mapping
  - `DeckManaCurve` - analyzed curve with stats
  - `ManaCurveRecommendation` - suggested adjustments
  - `StrategyCurveProfile` - ideal curves for aggro/control/midrange
- Helper functions:
  - `analyzeDeckManaCurve(deck)` - analyze current deck curve
  - `getManaCurveRecommendations(deckArchetype, currentCurve)` - get suggestions
  - `getStrategyProfile(archetype)` - get ideal curve for strategy
