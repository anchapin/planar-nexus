### Task 2.4.4: Add Missing Synergies Section
**Type**: auto
**Duration**: ~30 min

**Actions**:
1. Create `MissingSynergies` component:
```typescript
interface MissingSynergiesProps {
  missing: Array<{
    synergy: string;
    description: string;
    suggestion: string;
    impact: 'high' | 'medium' | 'low';
  }>;
}

export function MissingSynergies({ missing }: Props) {
  return (
    <Alert variant="warning">
      <AlertTitle>Missing Synergies</AlertTitle>
      {missing.map(m => (
        <AlertDescription key={m.synergy}>
          <strong>{m.synergy}:</strong> {m.description}
          <br />
          <em>Suggestion: {m.suggestion}</em>
        </AlertDescription>
      ))}
    </Alert>
  );
}
```

2. Style by impact level (high=red, medium=yellow, low=gray)
3. Place after present synergies

**Verification**:
- Missing synergies clearly marked

### Task 2.4.6: Add Export Coach Report
**Type**: auto
**Duration**: ~45 min

**Actions**:
1. Create export function:
```typescript
export function exportCoachReport(report: CoachReport, format: 'pdf' | 'text') {
  if (format === 'text') {
    const text = formatReportAsText(report);
    const blob = new Blob([text], { type: 'text/plain' });
    download(blob, 'deck-report.txt');
  } else if (format === 'pdf') {
    // Use browser print dialog for PDF
    window.print();
  }
}
```

2. Add export button to coach UI
3. Format report for readability:
