# Plan 2.3: AI Coach Synergy Analysis

## Objective
Implement comprehensive synergy detection that identifies card combinations and provides actionable recommendations.

## Why This Matters
- Users want to know which cards work well together (REQ-4)
- Synergy calls differentiate good decks from card piles
- Missing synergy detection helps deck improvement
- Foundation for advanced coach features

### Task 2.3.3: Implement Synergy Detection Algorithm
**Type**: auto
**Duration**: ~60 min

**Actions**:
1. Create `src/ai/synergy-detector.ts`:
```typescript
export function detectSynergies(deck: DeckCard[]): SynergyResult[] {
  const results: SynergyResult[] = [];
  
  for (const synergy of SYNERGY_DATABASE) {
    const match = checkSynergyMatch(deck, synergy);
    if (match.score > 0.5) {
      results.push({
        ...synergy,
        score: match.score,
        cardsInDeck: match.cards,
      });
    }
  }
  
  return results.sort((a, b) => b.score - a.score);
}
```

2. Implement matching logic:
   - Check for keyword presence
   - Check for tribal presence
   - Check for mechanic combinations
   - Calculate synergy score based on card count
   - Apply diminishing returns

3. Handle partial matches (e.g., 3 dragons without lord = 0.5 score)

**Verification**:

### Task 2.3.4: Add Missing Synergy Detection
**Type**: auto
**Duration**: ~45 min

**Actions**:
1. Implement "missing synergy" detection:
```typescript
export function detectMissingSynergies(
  deck: DeckCard[],
  archetype: string
): MissingSynergy[] {
  const missing: MissingSynergy[] = [];
  
  // Check archetype-specific synergies
  if (archetype === 'Dragon Tribal') {
    const hasLords = deck.some(c => c.name.includes('Dragonlord'));
    const dragonCount = deck.filter(c => c.type_line.includes('Dragon')).length;
    
    if (dragonCount >= 8 && !hasLords) {
      missing.push({
        synergy: 'Dragon Lords',
        description: 'You have 8+ dragons but no dragon lords',
        suggestion: 'Consider adding Dragonlord Kolaghan',
        impact: 'high',
      });
    }
  }
  
  return missing;
}
```

2. Create archetype-specific missing synergy checks
3. Generate actionable suggestions

**Verification**:
- Returns 1-3 missing synergies for typical decks
- Suggestions are specific and actionable
