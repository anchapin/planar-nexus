<task type="auto">
  <name>Task 1: Configure Jest coverage thresholds</name>
  <files>jest.config.js</files>
  <action>
1. Open jest.config.js
2. Add coverageThreshold configuration:
```javascript
coverageThreshold: {
  global: {
    branches: 60,
    functions: 70,
    lines: 70,
    statements: 70
  }
}
```
3. Ensure collectCoverage is set to true or can be enabled via --coverage flag
</action>
  <verify>npm test -- --coverage --passWithNoTests 2>&1 | grep -E "Coverage threshold"</verify>
  <done>Jest runs with coverage and enforces thresholds</done>
</task>

<task type="auto">
  <name>Task 2: Configure multi-format coverage reports</name>
  <files>jest.config.js</files>
  <action>
In jest.config.js, add coverageReporters configuration:
```javascript
coverageReporters: {
  "text-summary",
  "lcov",
  "html"
}
```
This generates:
- text-summary: Console output
- lcov: LCOV format for tools like Codecov/Coveralls
- html: HTML report in coverage/lcov-report directory
</action>
  <verify>ls -la coverage/lcov-report/index.html 2>/dev/null || ls -la coverage/</verify>
  <done>Coverage reports generated in lcov/ and html formats</done>
</task>

<task type="auto">

  <action>

2. Ensure there's a test step that runs coverage:

3. Add step to fail if coverage is below threshold (Jest will exit non-zero)
4. Optionally add artifact upload for lcov report
